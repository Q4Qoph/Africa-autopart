# Africa Auto Parts

A B2B platform connecting auto parts buyers and suppliers across Africa.

## Project Structure

```
.
├── src/
│   └── frontend/       # React + TypeScript + Vite web app
├── docs/               # Project documentation
├── tests/              # Cross-cutting test suites
└── .github/            # GitHub Actions workflows and templates
```

## Getting Started

### Frontend

```bash
cd src/frontend
npm install
npm run dev
```

The app connects to the backend API. Copy `.env.example` to `.env` and set `VITE_API_BASE_URL`.

## Tech Stack

| Layer    | Technology                         |
|----------|------------------------------------|
| Frontend | React 19, TypeScript, Vite, Tailwind CSS |
| UI       | shadcn/ui (Radix UI primitives)    |
| Forms    | React Hook Form + Zod              |
| HTTP     | Axios                              |
